# Café Real

O **Café Real** é um projeto desenvolvido como parte de uma atividade do curso de **Responsividade da PD EAD**.

O principal objetivo da atividade foi a construção de um site totalmente responsivo **sem a utilização de frameworks ou bibliotecas externas**, como Bootstrap ou similares. Todo o projeto foi desenvolvido utilizando **HTML, CSS e JavaScript puro**, aplicando conceitos de layout responsivo, organização de código e boas práticas de front-end.

## Sobre o projeto

O site apresenta a **Café Real**, uma empresa fictícia criada para o desenvolvimento da atividade. A marca foi idealizada como uma rede de cafeterias focada em cafés especiais, experiência do cliente e qualidade reconhecida.

Entre os serviços apresentados no site estão:

- Cafeterias físicas com ambientes confortáveis e acolhedores
- Planos de assinatura de cafés especiais, entregues diretamente ao cliente
- Destaque para premiações e reconhecimentos pela qualidade dos cafés

O projeto simula um site institucional, com seções informativas, chamadas para ação e estrutura pensada para se adaptar a diferentes tamanhos de tela.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Observações

Este projeto possui fins **exclusivamente educacionais** e não representa uma empresa real.
